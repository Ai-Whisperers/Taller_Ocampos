MECHANICS SHOP MANAGEMENT SYSTEM - EXCEL SETUP GUIDE
================================================================

STEP 1: IMPORT CSV FILES INTO EXCEL
-----------------------------------
1. Create a new Excel workbook
2. Import each CSV file as a separate worksheet:
   - 1_Clients.csv → "Clients" worksheet
   - 2_Vehicles.csv → "Vehicles" worksheet  
   - 3_WorkOrders.csv → "WorkOrders" worksheet
   - 4_PartsInventory.csv → "PartsInventory" worksheet
   - 5_ServicesCatalog.csv → "ServicesCatalog" worksheet
   - 6_FinancialTracking.csv → "FinancialTracking" worksheet
   - 7_Schedule.csv → "Schedule" worksheet
   - 8_Dashboard.csv → "Dashboard" worksheet

STEP 2: CONVERT TO EXCEL TABLES
-------------------------------
For each worksheet:
1. Select all data (Ctrl+A)
2. Insert → Table (Ctrl+T)
3. Ensure "My table has headers" is checked
4. Click OK
5. Name each table using the worksheet name (e.g., "Clients", "Vehicles", etc.)

STEP 4: CONDITIONAL FORMATTING
------------------------------

WORK ORDERS SHEET:
- Status column:
  * "Completed" = Green background
  * "In Progress" = Yellow background  
  * "Scheduled" = Blue background
  * "Waiting for Parts" = Orange background

PARTS INVENTORY SHEET:
- Quantity_OnHand column:
  * If <= Reorder_Point: Red background
  * If <= Min_Stock_Level: Orange background

FINANCIAL TRACKING SHEET:
- Payment_Status column:
  * "Paid" = Green background
  * "Pending" = Yellow background
  * "Overdue" = Red background
- Days_Overdue column:
  * If > 0: Red text

SCHEDULE SHEET:
- Priority_Level column:
  * "Emergency" = Red background
  * "Urgent" = Orange background

STEP 5: KEY FORMULAS TO ADD
--------------------------

VEHICLES SHEET:
Add column "Customer_Name":
=INDEX(Clients[First_Name]&" "&Clients[Last_Name],MATCH([@Client_ID],Clients[Client_ID],0))

WORK ORDERS SHEET:
Add column "Customer_Name":
=INDEX(Clients[First_Name]&" "&Clients[Last_Name],MATCH([@Client_ID],Clients[Client_ID],0))

Add column "Vehicle_Info":
=INDEX(Vehicles[Year]&" "&Vehicles[Make]&" "&Vehicles[Model],MATCH([@Vehicle_ID],Vehicles[Vehicle_ID],0))

PARTS INVENTORY SHEET:
Add column "Total_Value":
=[@Quantity_OnHand]*[@Cost_Price]

Add column "Reorder_Needed":
=IF([@Quantity_OnHand]<=[@Reorder_Point],"YES","NO")

FINANCIAL TRACKING SHEET:
Add column "Profit_Amount":
=[@Total_Amount]-[@Labor_Cost]-[@Parts_Cost]

Add column "Profit_Margin":
=[@Profit_Amount]/[@Total_Amount]

DASHBOARD SHEET:
Update formulas to reference actual table names:
- Daily Revenue: =SUMIFS(FinancialTracking[Total_Amount],FinancialTracking[Invoice_Date],TODAY())
- Monthly Revenue: =SUMIFS(FinancialTracking[Total_Amount],FinancialTracking[Invoice_Date],">="&EOMONTH(TODAY(),-1)+1)
- Work Orders Completed Today: =COUNTIFS(WorkOrders[Date_Completed],TODAY())
- Parts Below Reorder: =SUMPRODUCT((PartsInventory[Quantity_OnHand]<=PartsInventory[Reorder_Point])*1)

STEP 6: PIVOT TABLES FOR REPORTING
----------------------------------

CREATE THESE PIVOT TABLES ON NEW WORKSHEETS:

1. REVENUE BY SERVICE TYPE:
Source: FinancialTracking table
Rows: Description
Values: Sum of Total_Amount

2. CUSTOMER ANALYSIS:
Source: WorkOrders table  
Rows: Client_ID
Values: Count of Work_Order_ID, Sum of Total_Cost

3. TECHNICIAN PERFORMANCE:
Source: WorkOrders table
Rows: Technician
Values: Count of Work_Order_ID, Sum of Labor_Hours, Average of Labor_Hours

4. PARTS USAGE:
Source: WorkOrders table joined with PartsInventory
Rows: Parts_Used_IDs  
Values: Count of usage

5. MONTHLY TRENDS:
Source: FinancialTracking table
Rows: Invoice_Date (grouped by month)
Values: Sum of Total_Amount, Count of transactions
